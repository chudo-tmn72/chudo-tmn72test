ЧУДО — сайт v3 reference style

Эта версия повторяет выбранный референс без фейкового браузерного окна:
- розовый фон страницы;
- большая светлая витрина по центру;
- белая шапка;
- крупный hero-блок;
- блок «Подборки» с 3 карточками;
- преимущества;
- отдельный «Подбор по росту»;
- блок «Как заказать»;
- футер.

Что загрузить на GitHub:
1. index.html — в корень репозитория.
2. Папку images — в корень репозитория.

Минимально нужные картинки:
- images/hero-kids-clean.png
- images/logo-furry-chudo.png

Желательные картинки для карточек:
- images/cover-girls.png
- images/cover-boys.png
- images/cover-outfits.png

Если карточных картинок нет, сайт всё равно будет выглядеть нормально: останутся цветные градиенты.

Где менять дизайн:
В начале CSS есть блок:
ЧУДО — НАСТРОЙКИ ДИЗАЙНА

Там можно менять:
- --pink-main
- --pink-main-2
- --yellow
- --site-width
- --radius-xl
- --font-main
и другие переменные.

Где менять тексты:
В HTML есть комментарии:
- ШАПКА САЙТА
- ГЛАВНЫЙ БАННЕР
- ПОДБОРКИ
- СЕРВИСНЫЕ ПРЕИМУЩЕСТВА
- ПОДБОР ПО РОСТУ
- КАК ЗАКАЗАТЬ
- ФУТЕР

Кнопки сделаны кодом, не картинкой. Текст кнопок можно менять прямо внутри <a>...</a>.

Главный файл сайта должен называться строго:
index.html

Картинки должны лежать в папке:
images
